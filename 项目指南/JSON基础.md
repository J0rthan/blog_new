# Foundation

## JSON Types:

- String: "Hello world"
- Numbers: 10, -12, 0, 1e10
- Booleans: true, false
- null: null
- Arrays: [1, 2, 3], ["hello", "world"]
- Objects: {"key": "value"} - {"age": 30}

```json
{
    "name": "Jorthan",
    "age": 18,
    "isProgrammer": true,
    "hobbies": ["swimming", "eating"],
    "friends": [{
        "name": "Mike",
        "age": 19,
        "isProgrammer": true
    }]
}
```
